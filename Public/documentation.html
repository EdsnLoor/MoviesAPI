API Documentation

Below are the available endpoints for this API, along with their HTTP methods and descriptions.

Endpoint Method Body Description
/users GET N/A Retrieve a list of all users.
/movies GET N/A Retrieve a list of movies.
/movies/:Title GET N/A Retrieve a specific movie by their Title.
/genres/:Genre GET N/A Retrieve a description of a specific genre by their name.
/movies/genres/:Genre GET N/A Retrieve a list of movies by their genre.
/directors/:Name GET N/A Retrieve an information of a director by their name.
movies/directors/:Name GET N/A Retrieve a list of movies by their director.
/users/:Username GET N/A Retrieve a specific user by their Username.
/users POST { "username": "user5", "password": "123qwe", "email": "user5@test.com", "birthday": "1999-06-23" } Create a new user.
/users/:id PUT { "username": "Jose" } Update an existing user by their ID.
/users/:id/movies PUT { "movieID": "642b7d00df59c3bff5d4a750" } Update the FavoriteMovies list for a user by their ID.
/users/:id/movies DELETE { "movieID": "642b7d00df59c3bff5d4a750" } Deletes a movie by their id from the user's FavoriteMovies list.
/users/:id DELETE N/A Delete a user by their ID.
